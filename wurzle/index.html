How to Play Anleitung

There is a secret function behind the scenes and you need to find a root (i.e. an x where f(x)=0). You can input x values and you will be shown corresponding outputs. If the response is (approximately) zero, you win! The fewer values you need, the better your score. Es gibt eine geheime Funktion im Hintergrund und du musst eine Nullstelle finden (also einen Wert x, bei dem f(x)=0). Du kannst x-Werte eingeben und dir werden die entsprechenden Ausgaben angezeigt. Wenn das Ergebnis (annähernd) null ist, hast du gewonnen! Je weniger Eingaben du dazu brauchst, desto besser ist dein Ergebnis.

There is a new Wurzle every day. This Wurzle was created by .... You can suggest Wurzles here. Es gibt jeden Tag ein neues Wurzle. Dieses Wurzle wurde von ... erstellt. Du kannst selber Wurzles hier vorschlagen.

Pro Tips

  • The functions are made up of elementary operations like +-*/, trigonometric functions, sqrt, exponentials and polynomials. Consider inputting very large or small numbers to get an idea of how the function grows. Die Funktionen bestehen aus elementaren Operationen wie +, -, *, /, trigonometrischen Funktionen, sqrt, Exponential- und Polynomfunktionen. Überlege, sehr große oder sehr kleine Zahlen einzugeben, um eine Vorstellung davon zu bekommen, wie die Funktion wächst.
  • You only have a maximum of guesses, so try to use them wisely! Du hast höchstens Versuche, also nutze sie weise!
  • Sometimes, the root might not be rational. You can also input numbers like "pi" or "e*sqrt(2)". Manchmal ist die Nullstelle möglicherweise nicht rational. Du kannst auch Zahlen wie „pi“ oder „e*sqrt(2)“ eingeben.
  • A new Wurzle gets released every day at UTC+01:00. Professional Wurzlers wurzle as soon as possible to brag to their friends as soon as possible. Ein neues Wurzle wird jeden Morgen um UTC+01:00 (00:00 Uhr, Zentraleuropäische Zeit) veröffentlicht. Professionelle Wurzler wurzlen so früh wie möglich, um so früh wie möglich vor ihren Freunden angeben zu können.

Training Mode Übungsmodus

There is only one Wurzle per day. However, you can also activate Training Mode by clicking the button below! Training Mode will generate a random function for you to practice with. Be aware that these functions may not be as exciting as the daily ones, since those are hand-picked to be fun (they can be super hard!). Es gibt nur ein Wurzle pro Tag. Du kannst jedoch auch den Trainingsmodus aktivieren, indem du auf die Schaltfläche unten klickst! Der Trainingsmodus generiert eine zufällige Funktion, mit der du üben kannst. Beachte, dass diese Funktionen möglicherweise nicht so spannend sind wie die täglichen, da diese von Hand ausgewählt werden, um Spaß zu machen (sie können super schwer sein!).
