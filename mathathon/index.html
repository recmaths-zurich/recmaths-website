Mathathon

Mathathon is a team coding tournament where participants submit bots in Python or C++ to compete in four math-heavy strategy games. This prototype includes local testing in the browser, submission flows, and a mock leaderboard.

Mathathon ist ein Team-Coding-Turnier, bei dem Gruppen Bots in Python oder C++ für vier mathematisch geprägte Strategie-Spiele einreichen. Dieser Prototyp bietet lokales Testen im Browser, Einsende-Workflows und eine Mock-Rangliste.

Event Flow

Ablauf

1) Teams build a bot, 2) submit code, 3) backend runs matches, 4) leaderboard updates over rounds.

1) Teams bauen einen Bot, 2) Code einreichen, 3) Backend spielt Matches, 4) Rangliste wird laufend aktualisiert.

Game Slots

Spiel-Slots

Game A/B/C/D are placeholders for the final game set. Rules, input/output specs, and starter packs can be attached per game.

Spiel A/B/C/D sind Platzhalter für das finale Spielset. Regeln, I/O-Spezifikationen und Starter-Pakete können pro Spiel hinterlegt werden.

Scoring Draft

Punktesystem

Win = 3 points, Draw = 1 point, Loss = 0 points. Tie-breaks: average runtime, then direct encounters.

Sieg = 3 Punkte, Unentschieden = 1 Punkt, Niederlage = 0 Punkte. Tie-Break: mittlere Laufzeit, dann Direktduelle.
